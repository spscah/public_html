House Computing 2026

Answer as many questions as possible.

The number of marks for each question is shown on the right. Answer questions in any order.

For questions 1–5: show your answer to a referee, who will give you a code word to enter in the Google Form. For hidden questions: follow the instructions you find.

The first team to answer all questions correctly wins; otherwise the highest score after 30 minutes wins.

Question 1 [2 marks]

The world's first webcam was pointed not at a person or a street, but at a specific object.

At which university was it set up, and in what year was its image first made available on the internet?

Question 2 [2 marks]

The Collatz sequence starts with any positive integer. At each step: if the number is even, divide it by 2; if it is odd, multiply by 3 and add 1. Repeat until you reach 1.

Write a program to find which starting number below 1,000 produces the longest Collatz sequence. What is that number, and how many steps does it take to reach 1?

Question 3 [2 marks]

Write a program that accepts a positive integer (under a million) and outputs it in human readable form.

Test it with: 2026, 1999, 42 and 187,187.

Question 4 [2 marks]

Write a program that draws an analogue clock face showing the current time, with all three hands (hours, minutes, seconds) updating once per second.

Question 5 [2 marks]

Use an AI image generator to produce an image matching the following description as closely as possible. Show the result to a referee — marks are awarded for accuracy.

"A Victorian gentleman in a top hat, solemnly debugging a steam-powered computer while a confused penguin looks on."

Question 6 [2 marks]

Write a program that reads a plain-text file and outputs, in descending order of frequency, the 12 most commonly occurring words. Punctuation and capitalisation should be actively ignored (i.e. delete not replaced with spaces), but apostrophes and hyphens count as a letter.

Download this version of The Maltese Falcon and run your program on it.

Good luck. May the best team win.
